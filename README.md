# Texthooker UI

A fork that cleans up features that I don't need in texthooker UI because I'm a minimalist freak.

## Building

```
# Install pnpm
npm install --global pnpm

# Install dependencies
pnpm install

# Build the Page
pnpm run build

# Optional - serve the Page via local Server
pnpm run preview
```

The page can be opened via the index.html inside the "docs" folder and is usable without the need of hosting it via a server.
